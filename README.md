## 🚀 Featured Projects

### ⚡ [Power Outage Risk Analysis](https://github.com/Sejjjalll/power-outage-risk-dashboard)
> Python · Pandas · Plotly · EIA-861 · NOAA Storm Data

Combined IEEE utility reliability data (SAIDI/SAIFI) with NOAA storm records across **3,000+ US utilities** to build state-level risk rankings and an interactive choropleth dashboard. Identified **4 states** where outage performance was worse than storm exposure alone would predict. Fully automated from API pull to dashboard output — single script run.

`Python` `Plotly` `Choropleth` `SAIDI/SAIFI` `Risk Scoring` `API Automation`

---

### 🗺️ [Energy Infrastructure Analytics App](https://github.com/Sejjjalll)
> Python · GeoPandas · Folium · Census API · NREL API

Geospatial pipeline ranking **357 DFW energy assets** by solar potential, household reach, and EV opportunity. Built weighted scoring model (50% generation / 30% households / 20% EV gap) using Census ACS5 and NREL APIs with GeoPandas buffer joins. Outputs: ranked CSV, GeoJSON, Folium map, and Plotly dashboard.

`GeoPandas` `Folium` `Census API` `NREL API` `Weighted Scoring` `GeoJSON`

---

### 📦 [Delivery Insights: Drivers of Delays](https://github.com/Sejjjalll)
> Python · Pandas · SciPy · Scikit-learn · SQL · Tableau

Analyzed **197K delivery records** using statistical tests and logistic regression to uncover late delivery drivers. Found a **23% gap in on-time rates** across markets linked to demand-supply imbalance and peak-hour congestion. Evaluated single-variable interventions (only 3–5% improvement) — recommended dasher pre-positioning and routing optimization.

`Logistic Regression` `Hypothesis Testing` `Cohort Analysis` `Tableau` `Root Cause Analysis`
